Plenty of Spanish sentences need both pronouns at once: someone gives some*thing* to some*one*, and both the thing and the person are already known, so both get pronominalized in the same breath — "I gave it to her," not "I gave the book to María." [Chapter 12](../lesson-12) taught direct object pronouns and [Chapter 13](../lesson-13) taught indirect object pronouns as separate systems; this chapter is where they meet on the same verb. The good news is that almost nothing new has to be learned about placement — the infinitive/gerund/command rules are identical, just applied to a pair instead of a single pronoun. What's genuinely new is a fixed order the pair must appear in, and one small but memorable spelling change that keeps the pair pronounceable.

## Core Pattern: Combining Two Pronouns on One Verb

### Ordering: indirect object always comes before direct object
<LevelBadge level="A2" />

When a direct object pronoun ([Chapter 12](../lesson-12)) and an indirect object pronoun ([Chapter 13](../lesson-13)) both attach to the same verb, the indirect object pronoun always comes first. This order is fixed — it never flips, regardless of which specific pronouns are involved:

<SentenceStructure
  parts={[
    { text: "Te", role: "pronoun" },
    { text: "lo", role: "pronoun" },
    { text: "doy.", role: "verb" }
  ]}
/>

<ExampleSentence es="Te lo doy." en="I give it to you." note="te (indirect object, 'to you') before lo (direct object, 'it') — always in that order." />
<ExampleSentence es="¿Me la prestas?" en="Will you lend it to me?" note="me (indirect) before la (direct) — same fixed pattern regardless of which two pronouns are involved." />

<GrammarNote type="tip">
One memory hook: the person on the receiving end (indirect object) comes first, and the thing being received (direct object) comes second — which roughly matches the order English uses in "give it to me," not "give to me it."
</GrammarNote>

### The euphony rule: *le/les* becomes *se* before a direct object pronoun
<LevelBadge level="B1" />

There's exactly one wrinkle to the pattern above. Spanish never allows two consecutive pronouns that both start with *l-*. Since every direct object pronoun starts with *l-* (*lo, la, los, las*), an indirect object pronoun that would normally be *le* or *les* is forced to change to *se* right before it:

| Indirect object | + Direct object | Becomes |
|---|---|---|
| le | lo / la / los / las | se lo / se la / se los / se las |
| les | lo / la / los / las | se lo / se la / se los / se las |

<SentenceStructure
  parts={[
    { text: "Se", role: "pronoun" },
    { text: "lo", role: "pronoun" },
    { text: "doy.", role: "verb" }
  ]}
/>

<ExampleSentence es="Le doy el libro. → Se lo doy." en="I give him the book. → I give it to him." note="le would collide with lo, so it becomes se — never le lo." />
<ExampleSentence es="Les mandé la invitación. → Se la mandé." en="I sent them the invitation. → I sent it to them." note="les becomes se for the same reason, whether the underlying indirect object was singular or plural." />

<GrammarNote type="warning">
*Le lo* and *les lo* simply don't exist in Spanish — not as an error you might get away with, but as a combination the language doesn't permit at all. If you're ever tempted to write it, that's the exact signal to swap in *se*. This *se* is a pure spelling substitution for *le/les* — it isn't the reflexive *se* ([Chapter 15](../lesson-15)) or the passive/impersonal *se* ([Chapter 46](../lesson-46)), even though all three look identical on the page.
</GrammarNote>

## Building Out: Placement with Infinitives, Gerunds & Commands

The placement system is exactly the one [Chapter 12](../lesson-12) and [Chapter 13](../lesson-13) already taught — nothing new to learn there. The one habit to build is that the pair now moves as a single unit: wherever one pronoun goes, the other goes with it. They never split apart, with one attaching and the other staying in front.

### Two positions, together: infinitives and gerunds
<LevelBadge level="B1" />

Exactly as with a single pronoun, a conjugated verb paired with an infinitive or gerund allows the pronoun pair to attach to the end of the infinitive/gerund, or move as a unit in front of the conjugated verb:

<ExampleSentence es="Quiero dártelo. / Te lo quiero dar." en="I want to give it to you." note="dártelo — both pronouns fused onto the infinitive, with an accent added to keep the stress on the original syllable, same accent rule as Chapter 12." />
<ExampleSentence es="Voy a dárselo. / Se lo voy a dar." en="I'm going to give it to him/her." note="Same two-position choice, this time with le already converted to se before attaching." />

<GrammarNote type="tip">
Notice what doesn't happen: there's no *"Te quiero darlo"* or *"Dár lo te"* — you never see one pronoun attached and the other left behind. It's always both together, in both possible positions.
</GrammarNote>

### Mandatory attachment together: affirmative commands
<LevelBadge level="B1" />

The same affirmative/negative command contrast from [Chapter 12](../lesson-12) applies here — the pair attaches as a unit to an affirmative command and returns to the normal pre-verb position, still paired, with a negative command. The full command paradigm is [Chapter 36](../lesson-36)'s territory; what matters here is the pronoun-pair mechanics:

<SentenceStructure
  parts={[
    { text: "Diga", role: "verb" },
    { text: "me", role: "pronoun" },
    { text: "lo.", role: "pronoun" }
  ]}
/>
<SentenceStructure
  parts={[
    { text: "No", role: "connector" },
    { text: "me", role: "pronoun" },
    { text: "lo", role: "pronoun" },
    { text: "diga.", role: "verb" }
  ]}
/>

<ExampleSentence es="Dígamelo, por favor." en="Tell it to me, please." note="Dígamelo — usted command diga + me + lo, fused into one word with an accent added to preserve the original stress, the same rule Chapter 12 introduced for single pronouns." />
<ExampleSentence es="No me lo diga." en="Don't tell it to me." note="Negative command: both pronouns return to their normal pre-verb position, still in indirect-before-direct order." />

## Nuance, Exceptions & Contrast

### Common error: writing *le lo* instead of *se lo*
<LevelBadge level="B1" />

The single most common mistake at this stage is forgetting the euphony rule above and writing the two *l-* pronouns back to back:

<ExampleSentence es="Le lo di ayer." en="I gave it to him yesterday." note="Incorrect — le lo is not a real combination in Spanish. Should be Se lo di ayer." />

If you catch yourself writing *le lo* or *les lo/la/los/las*, that's the exact trigger to swap the indirect object pronoun to *se* — no other adjustment needed.

### Common error: reversing the order
<LevelBadge level="B1" />

The second common mistake is putting the direct object pronoun first, often because the English translation order ("give it to me") doesn't match Spanish's people-before-things logic as cleanly as it first seems:

<ExampleSentence es="Lo te doy." en="I give it to you." note="Incorrect order — direct object before indirect. Should be Te lo doy, indirect object first." />

Both errors are easy to catch with the same check: identify which pronoun is the person (indirect) and which is the thing (direct), put the person first, then confirm the person-pronoun isn't *le/les* sitting next to an *l-* direct object pronoun — if it is, convert it to *se*.

### *Se lo* can be just as ambiguous as *le* was
<LevelBadge level="B2" />

Because *se* here is standing in for both *le* and *les*, and for any of *él, ella, usted, ellos, ellas,* or *ustedes*, a sentence like *se lo di* is often genuinely ambiguous about who received the thing. Spanish resolves this exactly the way [Chapter 13](../lesson-13) resolved plain *le/les* ambiguity — by adding a clarifying *a* + prepositional phrase alongside the pronoun pair, not instead of it:

<ExampleSentence es="Se lo di a ella, no a él." en="I gave it to her, not to him." note="se alone couldn't distinguish 'her' from 'him' — a ella clarifies, exactly as Chapter 13's a-phrase strategy predicts, now applied to se." />

## Bringing It Together

<ExampleSentence es="Tenía dos entradas para el concierto. Quería dártelas, pero mi hermano me las pidió primero, así que se las di a él." en="I had two tickets for the concert. I wanted to give them to you, but my brother asked me for them first, so I gave them to him." note="The exact sentence Chapter 12 previewed. dártelas is te + las on an infinitive; me las pidió is me + las on a bare conjugated verb; se las di is le→se + las, converted because las is an l- pronoun. Every piece follows the ordering and euphony rules from this chapter." />

Working through any double-pronoun sentence comes down to three checks, in order: figure out which pronoun is the person and which is the thing (person goes first); check whether the person-pronoun is *le* or *les* sitting next to *lo/la/los/las* (if so, convert it to *se*); then apply Chapter 12 and 13's placement rules to the pair as a single, unsplittable unit.
